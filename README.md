## Getting Started

<!-- This is an example of how you may give instructions on setting up your project locally.
To get a local copy up and running follow these simple example steps. -->

### Prerequisites

1. First install [Python 3.10.11](https://www.python.org/downloads/release/python-31011/) or above

2. Install [pip](https://pip.pypa.io/en/stable/installation/)

    ```
      py https://bootstrap.pypa.io/get-pip.py
    ```

3. Choose one of the following solutions for adjusting the code if needed:
   
   * Install Jupyter Notebook
  
      ```
        pip install notebook
      ```
      
   * Install Visual Studio Code, click on this [link](https://code.visualstudio.com/)
   
### Installation 
1. Create a virtual environment of Python named env1

   ```
    python -m venv env1
   ```
   
2. Activate it
 
   * On Windows (Command Prompt or PowerShell)
     * For Command Prompt:

        ```
          env1\Scripts\activate
        ```   
     * For PowerShell:

        ```
          env1\Scripts\Activate
        ```   
   * On macOS/Linux (Bash/Zsh shell)

      ```
        source env1/bin/activate
      ```

3. Clone the repo

   ```
    git clone https://github.com/jorgegmayorgas/flask_iris
   ```

4. Install Python libraries

   ```
    pip install -r requirements.txt
   ```

5. Run application

   ```
    python app.py
   ```
